Selected Projects

Data Science

Assam Forest Cover Analysis

Python · Google Earth Engine · scikit-learn · LightGBM

End-to-end ML pipeline fusing multi-source satellite data (Landsat 8, ERA5 climate, Hansen Global Forest Change, SRTM) to classify forest pixels and track deforestation in Assam, 2015–2023. Engineered temporal features (lagged NDVI, rolling means, year-over-year deltas) and benchmarked Random Forest against LightGBM using leakage-safe temporal train/test splits.

View on GitHub

Global CO₂ Emissions Decoupling Analysis

Python · SQL (SQLite) · scikit-learn

Cleaned a 50,000-row, 79-column global emissions panel (Our World in Data) entirely in SQL, then classified 164 countries by economic–emissions decoupling (1995–2022) using window functions and surfaced distinct decarbonisation profiles with k-means clustering. Delivered as a reproducible Jupyter notebook.
